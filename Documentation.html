This plugin allows the developer using it to display a toast (temporary) message that will destroy itself in a given amount of time.

Importing

This plugin requires that jQuery be imported before the plugin itself (preferably in the <head> of the document). 

<head>
	<script src="http://code.jquery.com/jquery-2.0.3.min.js"></script>
	<script src="jquery.dpToast.js"></script>
</head>

Usage


Calling

Once imported the plugin can be called at any time. It is a simple method and does not need jQuery to select any dom elements. There are two basic ways that are suggested.

$.fn.dpToast("Hello World");
$().dpToast("Hello World");

Parameters

This method can take up to two parmeters, it may also only take one if desired.

message
Type: String
Description: This is HTML content that is placed in the toast message.
timeout
Type: Number
Description: This is the time (in milliseconds) that the message will be displayed. This is an optional argument.
Default: 3000 (3 seconds)

Example 1

Example 1 shows a basic toast, a <br> within the string is used to make it a multi-line message.

Show/Hide Code
<!DOCTYPE html>
<html>
<head>
	<title>Example 1</title>
	<script src="http://code.jquery.com/jquery-2.0.3.min.js"></script>
	<script src="jquery.dpToast.js"></script>
</head>
<body>
	<button onclick="$.fn.dpToast('Secret code: '+Math.round(Math.random()*1000)+'<br>This message will self destruct in 3 seconds.');">Toast</button>
</body>
</html>

Styling

Style the toast message using the class dp-toast. This class has inline styles so you will need to use !important in order to make the styles appear.

Example 5

Example 5 makes the background-color of the toast red.

Show/Hide Code
<!DOCTYPE html>
<html>
<head>
	<title>Example 5</title>
	<script src="http://code.jquery.com/jquery-2.0.3.min.js"></script>
	<script src="jquery.dpToast.js"></script>
	<style>
		.dp-toast {
			background-color: red !important;
		}
	</style>
</head>
<body>
	<button onclick="$.fn.dpToast('Secret code: '+Math.round(Math.random()*1000)+'<br>This message will self destruct in 3 seconds.');">Toast</button>
</body>
</html>
